# Исследование распределения логдоходностей

Исследование распределения логдоходностей финансовых временных рядов на примере самых популярных российских акций.

**Что сделано**:
- Проверка логдоходностей на нормальное распределение (Shapiro-Wilk, Jarque-Bera).
- Проверка логдоходностей на t-распреление Стьюдента (Kolmogorov-Smirnov)
- Сравнение частоты экстремальных колебаний (толстые хвосты) с нормальным и t-распределением.
- Сбор статистики по нескольким популярным российским акциям.

**Используемые технологии**:
- Python 3
- Pandas, NumPy
- Matplotlib
- SciPy, Statsmodels

**Вывод**:

Логдоходности финансовых инструментов демонстрируют ассиметрию и тяжёлые хвосты. 
t-распределение с низкими степенями свободы описывает риск более адекватно, чем нормальное распределение.
