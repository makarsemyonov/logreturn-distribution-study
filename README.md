# Исследование распределения логдоходностей

Исследование распределения логдоходностей финансовых временных рядов на примере самых популярных российских акций.

**Что сделано**:
- Проверка логдоходностей на нормальное распределение (Shapiro-Wilk, Jarque-Bera).
- Проверка логдоходностей на t-распреление Стьюдента (Kolmogorov-Smirnov)
- Проверка логдоходностей на распределение Лапласа
- Сравнение частоты экстремальных колебаний
- Сбор статистика по четырём популярным акциям: SBER, GAZP, LKOH, VTBR.

**Используемые технологии**:
- Python 3
- Pandas, NumPy
- Matplotlib
- SciPy, Statsmodels

**Вывод**:

t-распределение с низкими степенями свободы описывает риск наиболее адекватно.
